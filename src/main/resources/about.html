About SpeechDrop

What file formats are allowed?

You can upload DOC, DOCX, ODT, RTF, PDF, and TXT files that are up to 5MB.

Why should I use this over {insert competing service}?

SpeechDrop supports drag-and-drop, has auto-refreshing, preserves filenames, lacks ambiguous characters in room codes, looks better (especially on mobile devices), uses HTTPS, and most importantly, it has a cooler name and fancy animations.

How long are files kept for?

Files are deleted two weeks after room creation.

Which browsers are supported?

Modern versions of Chrome, Safari, Firefox, Edge, Opera, and IE11 will work.

What's this powered by?

This is hosted on a 32GB RAM, i7 4770k box from OVH. The backend uses nginx and pippo, and the frontend uses picnic.css and vue.js.

The source is available on GitHub.
